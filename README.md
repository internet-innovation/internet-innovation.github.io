# Internet Innovation Initiative

This is the website for the [Internet Innovation Initiative](https://internetequity.uchicago.edu) at the University of Chicago.

## About

The Internet Innovation Initiative develops tools and techniques for measuring Internet performance and connectivity. Our work focuses on device-based measurement using open-source software that runs on commodity hardware like Raspberry Pi devices.

## Site Structure

This site is built with Jekyll using the [al-folio](https://github.com/alshedivat/al-folio) theme.

### Pages

- **Home** (`/`): Main landing page with initiative overview
- **Projects** (`/projects/`): Research projects including FLOTO and Netrics
- **Data & Code** (`/datasets-code/`): Open datasets and software repositories
- **FLOTO** (`/floto/`): FLOTO project details and participation info

### Projects

- **FLOTO**: Large-scale edge device deployment for broadband research
- **Netrics**: Open-source Internet performance measurement system
- **Netrics Dashboard**: Web interface for visualizing measurements

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to view the site locally.
